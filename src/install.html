Installation Instructions

Windows

Download and run the Windows installer which will automatically install the native messaging host and mpv:

Download Windows Installer

The installer will:

  • Install the native messaging host
  • Download and install mpv
  • Configure everything automatically

Linux

Install using pip:

pip install --user open-in-mpv
open-in-mpv-install --user --force

Make sure you have mpv installed on your system. You can install it using your package manager: 

# Debian/Ubuntu
sudo apt install mpv

# Fedora
sudo dnf install mpv

# Arch Linux
sudo pacman -S mpv

macOS

Install using pip:

pip install --user open-in-mpv
open-in-mpv-install --user --force

Install mpv using Homebrew:

brew install mpv

Next Steps

  1. Follow the platform-specific instructions above
  2. Restart your browser
  3. Right-click on any video link or page and select "Open in mpv"
